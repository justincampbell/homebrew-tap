# justincampbell/homebrew-tap

Homebrew tap for Justin Campbell's command-line tools.

## Installation

```bash
brew tap justincampbell/tap
```

## Formulae

### use

Shell utility for managing multiple profile configurations.

```bash
brew install justincampbell/tap/use
```

[Repository](https://github.com/justincampbell/use)

### q-queue

Queue commands to run at a later time in this directory.

```bash
brew install justincampbell/tap/q-queue
```

[Repository](https://github.com/justincampbell/q-queue)

### cacheout

Caches output from commands for the specified timeframe.

```bash
brew install justincampbell/tap/cacheout
```

[Repository](https://github.com/justincampbell/cacheout)
